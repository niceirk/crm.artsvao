Техническое задание

Модуль «Расписание и подбор помещений»

1. Назначение модуля

Модуль предназначен для:

отображения полной картины занятости помещений учреждения;

планирования занятий, мероприятий, аренды, резервов;

поиска свободных помещений по заданным условиям;

подбора подходящего помещения по дате/времени/параметрам;

отображения активностей, которые идут прямо сейчас.

Модуль включает два глобальных режима:

Расписание

Подбор помещений

Переключение между режимами происходит через вкладки:

[ Расписание ]   [ Подбор помещений ]

2. Режим «Расписание»

2.1. Назначение

Обзор всех активностей по помещениям на конкретный день.

2.2. Фильтры режима

Сегодня (кнопка)

Дата (date-picker)

Помещения (мультивыбор или «все»)

Тип активности

Локация (если есть)

СЕЙЧАС (показать только текущие активности)

2.3. Вид отображения

Список помещений со списком активностей

Структура:

помещение

список активностей за день

свободные интервалы

подсветка текущих активностей при включенном фильтре «СЕЙЧАС»

3. Режим «Подбор помещений»

3.1. Назначение

Поиск помещений, удовлетворяющих заданным условиям.

3.2. Фильтры режима

Обязательные:

Дата

Временной интервал (в минутах)



Основные:

Помещение (конкретное помещение)

Только помещения со свободными слотами

Площадь/вместимость

Тип помещения



Дополнительные:

Локация / корпус / этаж

СЕЙЧАС (режим реального времени)

3.3. Вывод списка помещений

Каждая строка включает:

название помещения

статус доступности

первый подходящий свободный слот

краткий список активностей

4. Карточка помещения

Открывается при клике.

Содержит:

таймлайн на выбранную дату

список активностей

список свободных окон

кнопка «Создать активность в этом окне»

5. Правила пересечений

Принцип: ничего ни с чем не пересекается.
Любые две активности в одном помещении не могут пересекаться по времени.

6. Мобильная версия

отображается только режим «список помещений»

упрощённые фильтры

карточка помещения открывается на весь экран

кнопка создания активности доступна

7. Доп. UX

подсветка текущего времени

подсветка активностей, идущих сейчас

автообновление данных (опционально)

иконки типов активностей
