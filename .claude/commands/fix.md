# Исправление ошибок

Помоги найти и исправить ошибки в проекте:

1. **Сначала собери информацию:**
   - Запусти `npm run build` в backend и frontend
   - Проверь TypeScript ошибки с `npx tsc --noEmit`
   - Проверь логи pm2 если сервер запущен

2. **Проанализируй ошибки:**
   - Определи корневую причину
   - Найди связанные файлы

3. **Исправь последовательно:**
   - Начни с критических ошибок
   - Проверяй каждое исправление

4. **Финальная проверка:**
   - Убедись что сборка проходит
   - Протестируй затронутую функциональность
