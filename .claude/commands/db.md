# Работа с базой данных

Помоги с операциями базы данных Prisma:

## Доступные операции:
1. **Миграции**: создать, применить, откатить
2. **Схема**: проверить, сгенерировать клиент
3. **Данные**: просмотреть, отладить запросы

## Частые команды:
- `npx prisma migrate dev --name имя_миграции` — создать миграцию
- `npx prisma db push` — применить изменения без миграции
- `npx prisma generate` — обновить Prisma Client
- `npx prisma studio` — GUI для просмотра данных

Спроси какую операцию нужно выполнить.
