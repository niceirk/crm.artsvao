# Деплой проекта

Выполни деплой проекта на продакшен сервер:

1. Проверь текущий статус git (есть ли незакоммиченные изменения)
2. Запусти `./deploy-unified.sh` и следи за выполнением
3. После деплоя проверь health endpoint на сервере
4. Выведи краткий отчёт о результате деплоя

**Важно:** Перед деплоем убедись, что все тесты проходят и сборка успешна.
