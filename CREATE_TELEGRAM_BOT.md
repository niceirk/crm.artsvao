# Как создать Telegram бота для тестирования

## Шаги:

1. **Откройте Telegram** и найдите бота **@BotFather**

2. **Отправьте команду:** `/newbot`

3. **Введите название бота** (например): `АртсВАО CRM Test`

4. **Введите username бота** (должен заканчиваться на `bot`):
   - Например: `artsvao_crm_test_bot`

5. **@BotFather вернёт вам токен**, примерно такой:
   ```
   1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
   ```

6. **Скопируйте этот токен** и отправьте мне

## Что делать с токеном:

Я добавлю его в `.env` файл и настрою webhook для приёма сообщений.

---

## Альтернатива - использовать существующего бота:

Если у вас уже есть бот артсВАО, можно использовать его токен.
Токен можно получить у @BotFather командой `/mybots` → выбрать бота → API Token
